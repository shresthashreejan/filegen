# FILEGEN

**FILEGEN** is a simple command-line utility to generate a file with a specified number of lines.

## Requirements

- `make`
- `gcc`

## Usage

Build the executable:
```bash
make
```

Format:
```bash
./build/filegen <number-of-lines> <output-filename>
```

Example:
```bash
./build/filegen 100000 file.txt
```
